# pset-tex

My template for problem sets. Updated ad-hoc as I add to it.

Here is an example use of the package.
```
\documentclass[12pt]{article}
\usepackage[class={},prof={},title={},due={},tas={}]{pset}

\formatdocument{}

\begin{problem}
\end{problem}

\begin{solution}
\end{solution}

\begin{document}
  \makepset{}
\end{document}
```
